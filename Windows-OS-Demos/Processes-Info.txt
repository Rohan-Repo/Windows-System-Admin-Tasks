Windows:

List of all running processes:
tasklist

Filter By Name:

Notepad: tasklist | find "Notepad"

VSCode: tasklist | find "Code"

Calculator: tasklist | find "Calc"

Kill the Process:
/F for forcefully and /pid for specifying the ProcessID
taskkill /F /pid 15728

/F for forcefully and /im for specifying the Application Name
taskkill /F /im "CalculatorApp.exe"

Powershell:

List of all running processes:
Get-Process

Filter By Name:

Calculator: Get-Process | Select-String -Pattern "Calc"

Individual Process Info: Get-Process -Name "CalculatorApp"

Kill the Process: Stop-Process -Name "CalculatorApp"
